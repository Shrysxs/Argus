# Contracts

Solidity (or other language, per-chain if needed) smart contracts for the Argus on-chain registry.

The registry contract's job is minimal: accept a decision payload, emit an event, don't hold funds, don't gatekeep — see AGENTS.md §4.3.

> **No contract code yet.** Contract implementation is deferred until the chain decision is finalized (AGENTS.md §4.2, Phase 2).
